# 🔐 Generador Seguro de Contraseñas

## 📖 Descripción

El **Generador Seguro de Contraseñas** es un proyecto desarrollado para la asignatura **Lógica de Programación** de la Universidad Internacional del Ecuador (UIDE).

El sistema permite generar contraseñas robustas y personalizadas mediante la combinación de letras mayúsculas, letras minúsculas, números y caracteres especiales, aplicando estructuras condicionales, ciclos repetitivos y funciones desarrolladas en Python.

## 🎯 Objetivo del Sistema

Desarrollar una aplicación que permita generar contraseñas seguras de forma aleatoria, ofreciendo al usuario la posibilidad de configurar la longitud de la contraseña y seleccionar los tipos de caracteres que desea incluir, aplicando buenas prácticas de programación y organización del código.

---

## ⚙️ Funcionalidades

El sistema permite:

- Configurar la longitud de la contraseña.
- Seleccionar letras mayúsculas.
- Seleccionar letras minúsculas.
- Seleccionar números.
- Seleccionar símbolos especiales.
- Validar que el usuario seleccione al menos un tipo de carácter.
- Generar contraseñas aleatorias.
- Mostrar la contraseña generada.
- Permitir generar múltiples contraseñas durante la misma ejecución.

## ▶️ Ejecución del Programa

1. Instalar Python 3.
2. Descargar o clonar este repositorio.
3. Abrir el archivo **main.py** en Visual Studio Code.
4. Ejecutar el programa con el siguiente comando:

```bash
python main.py
```

o, dependiendo de la instalación:

```bash
python3 main.py
```

---

## 📋 Funcionamiento General

1. El usuario ingresa la longitud de la contraseña.
2. Selecciona los tipos de caracteres que desea utilizar.
3. El sistema valida la información ingresada.
4. Se genera una contraseña aleatoria utilizando los parámetros seleccionados.
5. El usuario puede generar una nueva contraseña o finalizar el programa.
